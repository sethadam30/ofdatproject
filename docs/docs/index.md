# OFDAT Documentation

For full documentation visit [ofdat.org](http://ofdat.org).

## Documentation commands

* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs help` - Print the help message.
